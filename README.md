# Personal Portfolio Website

A personal portfolio website showcasing my work, publications, and interests.

## Getting Started

1. Clone this repository
2. Customize the content in `index.html` with your information
3. Add your images to the `images/` folder
4. Deploy to GitHub Pages

## Deployment to GitHub Pages

1. Go to your repository Settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select the branch (usually `main`)
4. Click Save
5. Your site will be live at `https://yourusername.github.io/repository-name`

## Customization

- Update text content in `index.html`
- Modify colors and fonts in `css/style.css`
- Add your own images to the `images/` folder
- Customize animations and interactions in `js/main.js`
